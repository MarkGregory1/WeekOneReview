How to Push to GitHub

PART ONE

Move into the correct folder using the cd command in Terminal

If it is the first time pushing to GitHub, type the command git init (you will know this is successful if you see git:(master) after the folder name

Type the command git add . (to add all files within the folder, otherwise you can add specific files by typing file names after add)

Type the command git commit -m"" with any message inside the ""

PART TWO

1. Go to GitHub.com

2. Click the green button that says NEW to start creating a new repository

3. Name your repository (In this case, (WeekOneReview) then scroll down and click Create Repository (the next green button)

4. Copy the 3 lines of Git commands

PART THREE

1. Return to VS Code

2. Paste the 3 lines from GitHub into Terminal

3.Wait for it..

4.Go back to GitHub

5.Refresh the page

Voiloa!
